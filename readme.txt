Unofficial diagnostic tool (beta) for the official Google Publisher Plugin (beta).

== Description ==

This plugin shows you some more-less relevant settings and parameters from Wordpress installation, if you have any issues with official Google Publisher Plugin plugin.

All you can see above on "GPP Diag (unofficial)" Options page, is all this plugin is doing, it doesn't make any change, it doesn't affect your Wordpress installation in any way, nor ad serving.

If you don't need this plugin - uninstall it. You can always install it back, when and if needed.

Standard procedure - as any other WP plugin.
